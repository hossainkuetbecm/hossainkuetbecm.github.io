Structural Project

Syed Abdul Alim Tuhin — SB+09 Storied Residential / Mixed-Use Building

RCC SB+09 storied residential/mixed-use building documentation support.

Project Overview

This project involved preliminary structural drawing and drawing index support for a semi-basement plus 09-storied residential / mixed-use RCC building located at North Kattoli, CDA Residential Area, Akber Shah, Chattogram.

My Responsibilities

  • Prepared preliminary structural drawing and structural drawing index.
  • Prepared general notes, site layout plan, column layout plan, column schedule and shear wall schedule.
  • Prepared grid-to-pile dimension layout, pile long section details, pile cap layout by zone and pile/pile cap details.
  • Prepared UGWR layout, underground water reservoir details, retaining wall layout/details and septic tank layout/details.
  • Prepared ramp and stair details.
  • Prepared semi-basement roof beam and slab layout.
  • Prepared typical floor roof beam layout, typical floor roof slab details, roof top plan and OHWR/lift machine room details.

Work Scope

  • Preliminary structural drawing
  • Drawing index preparation
  • AutoCAD drafting support
  • Foundation and pile documentation
  • RCC building documentation coordination
