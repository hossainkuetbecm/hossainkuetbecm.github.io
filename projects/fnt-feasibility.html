Technical Report

Fateyabad New-Township Feasibility / Inception Report Support

Technical documentation and report support for proposed Fateyabad New-Township Development Project.

Project Overview

This project involved technical documentation and report support for the Inception Report / Feasibility Study of Fateyabad New-Township Development Project, covering technical, economic, social, environmental, master planning and infrastructure components.

My Responsibilities

  • Prepared inception report formatting, document lead sheet, project synopsis, background, project brief, project organization, personnel deployment and work program.
  • Documented project objective to reduce housing pressure, provide planned urban facilities, create residential/commercial opportunities and develop a self-sustained township with modern facilities.
  • Prepared site appreciation including project land area, existing structures, roads, water bodies and guided development concept.
  • Supported scope of evaluation including survey report study, soil test report study, conceptual master plan, urban plan, structural plan and transportation plan.
  • Supported storm-water/drainage plan, conceptual layout drawings and simulation-animation scope documentation.
  • Supported infrastructure development documentation including road, pavement, intersection design, bridge, culvert, drainage and utilities.
  • Assisted guided development design description, stakeholder analysis, possible investors, transportation facility, socio-economic analysis, environmental analysis, traffic assessment, preliminary cost estimate and BOQ.

Work Scope

  • Inception report support
  • Feasibility study documentation
  • Technical report formatting
  • Infrastructure development documentation
  • Preliminary cost estimate and BOQ support

Project Information

Year: April–June 2022

Location: Chattogram, Bangladesh

Client / Authority: Chittagong Development Authority (CDA)

Role: Technical Documentation and Report Support

Category: Planning
