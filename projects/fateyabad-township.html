Fateyabad New Township Documentation

Documentation, presentation and estimation support for a proposed smart and self-sustained township.

Project Overview

This project involved documentation, presentation and estimation support for Development of Fateyabad New Town / Park Town Fateyabad New Township. The proposed township included residential, commercial, government, public amenities, infrastructure, utilities, open space, civic/cultural and guided development components.

My Responsibilities

  • Prepared PowerPoint project presentations explaining vision, location, objectives, land area, project features and development components.
  • Included smart city concept, city-in-a-garden concept, neighborhood planning and housing for all income groups.
  • Prepared presentation content for education village, waste treatment plant, sports complex, medi-village, commercial area, administrative area, diplomatic enclave, focal point and public transportation/LRT concepts.
  • Prepared land-use and development feature presentation covering government, residential, commercial, public amenities, infrastructure/utilities, open space and civic/cultural components.
  • Prepared cost component presentation for infrastructure development and guided development.
  • Prepared revenue concept including land sale, flat/commercial space sale, tourism/hotel/restaurant activities, betterment fee and operation/maintenance transfer concept.
  • Supported development procedure presentation including feasibility study, TOR, MOU, G2G, PPP/EPC, BOT and project implementation steps.
  • Prepared video/visual presentation support for project communication and client presentation.

Work Scope

  • Township presentation
  • Planning documentation
  • PowerPoint and visual presentation support
  • Cost component documentation
  • Land-use and guided development data presentation

Project Information

Year: 2021–2022

Location: Fateyabad, Hathazari, Chattogram

Client / Authority: Chittagong Development Authority (CDA)

Role: Documentation, Presentation and Estimation Support

Category: Planning
